# generator-hw

Yeoman generator for creating code projects

## Prerequisite

```bash
$ npm i yeoman -g
```

## Usage

```bash
$ yo hw --help
```

## Provided templates

### node

A basic nodejs project with mocha tests and linter

### node-ts

A nodejs + typescript project with mocha tests and linter

Enjoy!
